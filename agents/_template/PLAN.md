# Plan — _template

Describe the agent's approach. This file is for you and reviewers — it's
not surfaced to the model. Use it to pin down:

1. **Objective** — what the agent is for in one sentence.
2. **Inputs** — what the agent expects the user to provide.
3. **Outputs** — what the agent produces and where it puts them.
4. **Tools used** — why each tool is needed (and why others aren't).
5. **Safety boundaries** — what the agent must never do.
6. **Open questions** — what's unresolved.
